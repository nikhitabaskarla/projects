Project-1

Facial Recognition & Emotion Detection System

Overview

This project focuses on building real-time system capable of detecting human faces and recognizing facial emotions using classical computer vision and deep learning techniques. It combines the Haar Cascade Classifier for efficient face detection and a CNN model trained on the FER2013 dataset for emotion classification.

Project-2

Hosting Website using EC2 Instance

Overview

Deployed a static website on an AWS EC2 instance using Ubuntu. Installed and configured the Apache web server to serve web pages. Used WinSCP to securely transfer a free HTML website template to the server.
